DQ · Displaceability Quotient 评估框架 v1.1

AI可替代岗位评估打分卡

从四个维度量化评估每个岗位的AI替代风险，附带打分卡模板与行动建议

📊 使用说明
1. 为组织架构中的每个关键岗位填写本打分卡（建议由HRBP + 直属上级共同评分）
2. 每题打分 1-5（1=完全同意/高度符合，5=完全不同意/不符合）
💡 评分锚点：1分 = 该描述在每日工作中出现频率 ≥80%；5分 = 该描述几乎不发生（≤10%）
  维度①正向计分（分低=易替代）；维度②③④反向计分（分低=难替代，由系统自动翻转） 3. 维度得分 = 该维度下所有题目平均分（维度②③④由系统自动反向计分）。
  总分 DQ = 四个维度得分的加权平均值（权重：①30%、②25%、③25%、④20%）
4. DQ < 2.0 → 🔴 高风险，建议12个月内启动AI替代计划
5. DQ 2.0–3.5 → 🟡 中风险，建议混合模式（AI辅助 + 人工审核）
6. DQ > 3.5 → 🟢 低风险，重点放在能力提升而非替代
